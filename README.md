## Leetcode structure

1. Read the problem aloud
2. Ask a minimum of 1 clarifying question.
   - What's the input?
   - Any constraints? Space? Time?
3. Ask for sample input and confirm the expected results
4. Pseduo code out the steps to solve problem
5. Confirm you are on right track before coding
6. Code while talking through your through process
7. Walk through the example with your code, line by line
8. Big O + any data structures needed for your solution
9. Talk about possible improvements
